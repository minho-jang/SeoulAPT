# Seoul APT

서울특별시 아파트 매매 실거래 정보를 한 눈에!

## Features

- 공공데이터 오픈API : 국토교통부 아파트매매 실거래자료
  - 실거래자료 DB 업데이트 자동화
- SPA (Single Page Application)
  - Vue.js + SpringBoot + MySQL
- Kakao Map API
  - Map, Marker, Clustering, Roadview
